# Classes Vs Files section

This class contains tests to validate the rule for variable declaration and identifiers

1. A valid identifier must start with a letter, $ or _ (underscore). Can include numbers but not at the start of the identifier itself.
2. If an illegal combination of identifier is used, the file do not compile
3. In Java, we can declare multiple variables at once with a specific syntax
4. It is illegal to define more than one type, more than once, in a multiple declaration statement
